FutureTank — نسخة جاهزة للنشر (Static)
المحتويات:
- index.html (الصفحة الرئيسية مع متجر وخدمات وسلة)
- assistant.html (المساعد الذكي)
- billing.html (إنشاء الفواتير وتنزيل PDF)
- dashboard.html (لوحة تحكم محليّة - default password: future2025)
- styles.css (الأنماط)
- database.json (قالب البيانات)
تثبيت:
انسخ المجلد إلى مستودع GitHub في الفرع main (root). بعد الرفع افتح Settings → Pages وتأكد أن النشر من branch: main, folder: /(root).
